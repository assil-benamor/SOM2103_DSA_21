## Steps

1. Clone the repo
   ```sh
   git clone https://github.com/lilos404/SOM2103_DSA_21
   ```
   
2. Add the clean data set under input/


3. Run the aggregation
   ```r
   source("src/1_aggregation.R")
   ```
